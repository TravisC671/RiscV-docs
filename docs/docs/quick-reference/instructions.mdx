# AArch 64 Instructions

<details>
    <summary>**adc** - add with carry</summary>

    **pg. 86**
    ## Syntax

    `addc{s} Rd, Rn, Rm` <br />
    **\{s\}** (optional) determines if the operation affects the PSTATE<br />
    **Rd** is the destination register where the sum is stored<br />
    **Rn Rm** are the registers to be added together<br />
</details>
